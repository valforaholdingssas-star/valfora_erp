# Manual de usuario — Seeds ERP (resumen)

## Acceso

1. Abre la URL del sistema (proporcionada por tu administrador).
2. Inicia sesión con **correo** y **contraseña**.
3. Si olvidaste la contraseña, usa el flujo de recuperación desde la pantalla de login (si está habilitado).

## Roles

- **Super admin / Admin:** gestión de usuarios, CRM completo, configuración de IA, chat.
- **Collaborator:** operación en CRM y chat según permisos asignados.

## Módulos principales

### Panel principal

Muestra un resumen (contactos, deals abiertos, conversaciones, notificaciones pendientes).

### CRM

- **Contactos:** listado, filtros, alta/edición y ficha con pestañas (información, actividades, deals, documentos, historial).
- **Pipeline:** tablero Kanban de deals por etapa.
- **Dashboard CRM:** métricas del módulo (desde el menú CRM).

### Chat

- Lista de conversaciones y mensajes en tiempo real.
- Enlace desde la ficha de un contacto para abrir el chat asociado.
- **Modo IA:** puede activarse o desactivarse por conversación (según política de la empresa).

### Configuración de IA

- Pantalla para administradores: ajuste de modelo, prompt del sistema, límites y opciones de RAG (según permisos).

### Notificaciones

- Icono en la cabecera: lista de avisos (p. ej. mensajes nuevos de clientes) y opción de marcar como leídas.

## Soporte

- Incidencias de **WhatsApp** o **correo** empresarial: contacta al administrador de la plataforma.
- Errores de sistema: anota la hora, la pantalla y el mensaje mostrado (sin capturas de datos sensibles).
